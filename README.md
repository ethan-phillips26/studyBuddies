# StudyBuddies

## Angular Setup

### Prerequisites
- Node.js
- npm

### Run Angular Server
```bash
ng serve
```

---

## Django Setup

### Prerequisites
- Python 3.8+ installed
- pip

### Install Dependencies
```bash
npm install
```

## Create Virtual Environment
```bash
python -m venv venv
```

## Activate Virtual Environment

* Windows
  ```bash
  venv\Scripts\activate
  ```
* Linux / MacOS
  ```bash
  source venv/bin/activate
  ```

## Install Dependencies
```bash
pip install -r requirements.txt
```

## Run Django Server
```bash
python manage.py runserver
```
